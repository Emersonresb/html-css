Principais Formatações

Colocar frase em negrito / destaque

Para colocar em negrito, usamos a tag B, por exemplo: Usando a tag b (forma não semantica)

Ou usamos a tag STRONG (forma semantica) exemplo: Usando a tag STRONG

Colocar frase em itálico

Vamos colocar um termo em itálico, por exemplo: Termo em itálico

agora vamos usar outra tag para itálico, a tag EM (forma semantica) exemplo: Usando a tag EM

Tag MARK

Tem uma tag chamada mark que serve como se fosse um marca texto

Texto grande e pequeno

Para texto grande e pequeno vamos usar duas tags , small e big, porém BIG é forma e sera usado futuramente em CSS já o small é semantica e pode ser usado no proprio HTML

Texto deletado

podemos marcar um texto como excluido para indicar que ele sera lido, mas nao considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois
